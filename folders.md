    /      -> Root Directory
    /bin   -> User Binaries
    /sbin  -> System Binaries
    /etc   -> Configuration Files
    /dev   -> Device Files
    /proc  -> Process Information
    /var   -> Variable Files
    /tmp   -> Temporary Files
    /usr   -> User Programs
    /home  -> Home Directories
    /boot  -> Boot Loader Files
    /lib   -> System Libraries
    /opt   -> Optional add-on Apps
    /mnt   -> Mount Directory
    /media -> Removable Devices
    /srv   -> Service Data
